# Mist

## setup dev envirvonment

### using `fyrer` (recommended)

- Install `fyrer`: 

```bash
  cargo install fyrer
```

- Clone the repository and start the development environment:

```bash
git clone https://github.com/corecollectives/mist
cd mist
fyrer
```

### using docker-compose
```bash
git clone https://github.com/corecollectives/mist
cd mist
docker-compose up --build
```
